如何在markdown中添加分隔线

  • 答:在一行中用三个以上的星号(*)、减号(-)、下划线(_)来建立一个分隔线；除空格外行内不能有其他字符；（除第一个符号的左侧最多添加三个空格外）三个相同符号两侧可以添加任意多个空格。

如何原样显示*,#这样的符号

  • 答:加反斜线\,比如*,\,#,\$,#,$,

有时候不加也可以显示正确,但有时候不能,所以最好都正规书写:\,\$,#,*,\$,#,*,\
