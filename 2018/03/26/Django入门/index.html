2 相关命令

  • 创建django项目: django-admin startproject projectName
  • 为django项目添加一个应用app: python manage.py startapp appName
  • 运行服务器: python manage.py runserver 8088 (自己指定的端口号)
  • 把自己的应用注册到项目: 在projectName/projectName/settings.py中的INSTALLED_APP键中添加对应的值’appName’
  • 生成迁移文件(根据模型类生成sql语句): python manage.py makemigrations
  • 执行迁移(执行sql语句生成数据表): python manage.py migrate
  • 进入django的shell: python manage.py shell
  • 创建后台管理员: python manage.py createsuperuser
