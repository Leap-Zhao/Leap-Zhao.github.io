sublime快捷键

发表于 2018-03-26 | 更新于: 2018-03-26 | 分类于 IDE ， sublime

总述

记录在sublime下一些常用的快捷键

1 基本常用键

  • 切换标签 : ctrl+tab
  • 切换到第n个标签栏 : alt+n ,比如切换到第4个标签栏alt+4
  • 将当前行往上移一行 : ctrl+shift+上箭头
  • 复制当前行到下一行 : ctrl+alt+下箭头

2 分屏模式下

  • 分n个屏 : alt+shift+n ,比如分2个屏alt+shift+2
  • 关闭分屏,即1个屏 : alt+shift+1
  • 将当前文件从当前屏放到屏n : ctrl+shift+n, 比如,放到屏3,就是ctrl+shift+3
  • 分屏模式下快速切换左右两边窗口 : Ctrl+对应窗口数字键
